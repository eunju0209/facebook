# React로 Facebook 댓글 기능 구현하기

## 기능
1. 댓글을 입력할 수 있습니다.
2. 댓글에 답글을 입력할 수 있습니다.
3. 댓글에 좋아요 누르고 취소할 수 있습니다.
4. 댓글을 삭제할 수 있습니다.
5. 입력한 데이터는 localStorage에 저장되어 새로고침 하거나 사이트에 다시 접속해도 기존 데이터가 삭제되지 않습니다.
